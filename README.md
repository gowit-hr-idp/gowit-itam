# IT 통합 자산관리 시스템 v3.0

경영지원팀 전용 IT 자산·정기결제·판촉물 통합 관리 플랫폼

## 🔑 로그인 / 회원가입

### 기본 제공 계정 (이메일 = 로그인 ID)
| 이메일 | 비밀번호 | 직책 |
|--------|----------|------|
| admin@gowit.co.kr | admin1234 | 팀장 |
| manager@gowit.co.kr | manager1234 | 파트장 |
| user01@gowit.co.kr | user1234 | 팀원 |

> 로그인 입력 시 `@gowit.co.kr` 앞자리(`admin`, `manager`, `user01`)만 입력

### 회원가입 규칙
- **gowit.co.kr 이메일 = 로그인 ID** (이메일이 곧 계정)
- 이메일 앞자리만 입력 → `@gowit.co.kr` 자동 결합
- 이름(실명) + 직책(팀장/파트장/팀원) + 비밀번호 입력
- 비밀번호: 8자 이상, 영문 + 숫자 필수, 강도 바 실시간 표시
- 이메일 중복 가입 방지 (DB 실시간 확인)
- 가입 즉시 로그인 가능

> 접속 가능 직책: **팀장 / 파트장 / 팀원**

## ✅ 구현 완료 기능

### 🔐 인증 시스템
- 로그인/로그아웃 (users DB 기반 인증)
- 세션 관리 (sessionStorage)
- 인증 가드: 미로그인 시 login.html 강제 이동
- 로그인 사용자 이름/직책 헤더·사이드바 표시
- 처리자 필드 자동 채우기 (로그인 사용자명)
- 아이디 저장 (localStorage)

### 🖥️ 고정자산 관리
- **자산 대장**: 전체 목록, 필터(분류/상태/부서/검색), 페이지네이션
- **자산 입고 등록**: 신규 등록 (중복 자산번호 방지)
  - PC/서버 분류 선택 시 **CPU / MEM / SSD** 필드 자동 노출
  - 기타 분류: 일반 사양 필드 표시
- **자산 수정**: CPU/MEM/SSD 포함 전체 필드 수정 가능
- **자산 상세 보기**: CPU/MEM/SSD 전용 섹션 포함
- **반출 관리**: 반출 중인 자산 목록 + CPU/MEM/SSD 표시 + 반납 처리
- **반납 관리**: 창고 반납 자산 + CPU/MEM/SSD + 재배출/폐기/매각
- **수리 관리**: 수리 중인 자산 + CPU/MEM/SSD + 수리완료 처리
- **폐기/매각**: 처리 대상 자산 + CPU/MEM/SSD + 처리 완료 목록
- **변경 이력**: 모든 상태 변경 이력 자동 기록 (검색/필터)
- **보증 만료 관리**: 만료/30일/90일 이내 경보 + CPU/MEM/SSD 표시
- **Excel 내보내기**: CPU/MEM/SSD 컬럼 포함

### 💳 IT 정기결제 관리
- **구독 목록**: 필터(카테고리/주기/상태) + 월 비용 합계 배지
- **구독 등록/수정/삭제**: 통화(KRW/USD/EUR/JPY) + 수기 환산 입력
- **갱신 알림**: 만료/30일/90일 이내 구분
- **비용 분석**: 카테고리 도넛 차트, 주기별 막대 차트, TOP10 순위
- **대시보드 탭**: 월 비용/연 비용/갱신 임박 통계

### 🎁 판촉물 관리 (신규)
- **재고 현황**: 전체 목록, 재고/분류/검색 필터, 빠른 입고/출고 버튼
  - 재고 없음: 빨간색 / 재고 부족(안전재고 이하): 주황색 / 정상: 초록색
- **입고 처리**: 품목 선택 → 수량/일자/공급처/담당자 입력 → 재고 자동 업데이트
- **출고 처리**: 품목 선택 → 수량/대상 고객사·행사명/담당자 입력 → 재고 자동 차감
  - 재고 초과 출고 방지
- **입출고 이력**: 전체 이력 조회 + 구분(입고/출고/수량조정/폐기) 필터
- **품목 등록/수정/삭제**: 코드, 품명, 분류, 규격, 단가, 안전재고, 초기재고 등록
- **대시보드 탭**: 총 품목/정상재고/재고부족/총 재고금액 + 분류별 차트 + 부족 품목 리스트
- **Excel 내보내기**

### 📊 통합 대시보드
- **고정자산 탭**: 상태별 통계, 도넛·막대 차트, 최근이력, 보증만료 알림
- **정기결제 탭**: 비용 통계, 카테고리/주기 차트
- **판촉물 탭**: 재고 현황, 분류별 차트, 재고부족 목록

## 📝 사용 가이드

### 자산 등록 시 CPU/SSD/MEM 입력 방법
1. 자산 분류 → **PC/노트북** 또는 **서버** 선택
2. "💻 PC/서버 사양" 섹션 자동 표시
3. CPU / MEM / SSD 각각 자유 형식으로 입력  
   예) CPU: `Intel i7-1360P`, MEM: `16GB DDR5`, SSD: `512GB NVMe`

### 판촉물 출고 시
1. **출고 처리** 메뉴 진입
2. 판촉물 선택 → 현재 재고 자동 표시
3. 출고 수량 입력 (재고 초과 시 오류 안내)
4. 고객사/행사명, 담당자 입력 후 처리

### 정기결제 외화 입력
- 통화 필드: USD/EUR/JPY 원본 금액 기재
- 월/연 환산 금액(원): 결제 당시 실제 환율로 수기 입력
